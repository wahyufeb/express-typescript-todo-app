# express-typescript-todo-app
Build Rest API with Express Typescript

### Dependencies
* ✅ bcrypt
* ✅ compression
* ✅ cors    
* ✅ dotenv
* ✅ express
* ✅ express-validator
* ✅ helmet    
* ✅ jsonwebtoken
* ✅ mysql2    
* ✅ sequelize    
* ✅ sequelize-cli

```bash
# Install Dependencies
$ npm install

# Compiling Typescript
$ npm run ts

# Starting project in development mode
$ npm run dev

```
